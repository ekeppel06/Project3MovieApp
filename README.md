# Project 3: Movie Voting App 🍿🎬

This is an app you can use with your friends to figure out what movie you guys want to watch! Just create a room, and have your friends join using the 6-digit generated code.

Once you're in a room, just search for the movie you want and add it to the list. Once you have a good collection, the host can start the voting process and everyone in the room can vote on a movie. A winner will be declared once the host ends voting.
